Proyectos recientes

Predicción de Incumplimiento de Préstamos mediante Redes Neuronales Artificiales (ANN).

Este estudio de caso abarca todo el proceso de aprendizaje automático en un intento de producir modelos útiles con Redes Neuronales Profundas (DNN) que puedan ser utilizados en la industria de préstamos. El estudio se centra en minimizar el riesgo financiero para la empresa prestamista. Mediante un Análisis Exploratorio de Datos (EDA) detallado, examinamos las relaciones entre las variables. En la etapa de preprocesamiento, transformamos los datos y realizamos ingeniería de características para mejorar las predicciones de nuestro modelo.

Pronóstico del tipo de cambio EUR/USD con simulaciones de Monte Carlo.

Este proyecto utiliza el Modelo de Movimiento Browniano Geométrico (GBM) para predecir la evolución futura del tipo de cambio EUR/USD. El GBM es un proceso estocástico comúnmente utilizado en modelado financiero para simular el movimiento aleatorio de activos como precios de acciones, tipos de cambio y precios de productos básicos. Este proyecto se desarrolló en un escenario real de gestión de riesgos para evaluar el riesgo que un cliente tenía en su cartera de derivados.
